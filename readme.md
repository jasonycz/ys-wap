# 玉石项目-移动端

## 安装
```
npm i -g gulp-cli    #安装全局gulp
npm i           #安装依赖
gulp            #启动开发环境
gulp build      #编译
npm run online  #发布到服务器
```

## 安装环境
```bash
npm i --save-dev gulp gulp-autoprefixer browser-sync gulp-load-plugins del run-sequence gulp-if gulp-concat gulp-angular-templatecache gulp-rev gulp-rev-collector gulp-uglify gulp-csso gulp-rimraf gulp-rename gulp-clean-css gulp-imagemin gulp-angular-filesort gulp-util gulp-plumber gulp-sass shipit-deploy  
```

## 安装依赖js
```
npm i --save angular angular-animate angular-cookies angular-resource angular-route angular-sanitize angular-touch ngstorage moment jquery angular-i18n material-design-iconic-font font-awesome weather-icons  
```
